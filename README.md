# Personal Portfolio Website

This is a simple **personal portfolio website** created using **HTML and basic CSS**. The website contains two main pages: a **Home page** and a **Contact page**.

The **Home page** introduces my profile and provides a basic overview of the website, while the **Contact page** allows visitors to reach out to me.

This project was built to practice the fundamentals of **HTML structure and CSS styling** while creating a clean and simple website layout.

## Technologies Used

* HTML5
* CSS3

## Pages

* **Home** – Main introduction page
* **Contact** – Page for contacting me

## Purpose

The goal of this project is to practice basic **front-end web development** and understand how to structure a multi-page website using HTML and CSS.
